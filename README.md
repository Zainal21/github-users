## Installation

1. Clone repository

```bash
$ git clone https://github.com/Zainal21/github-users.git
```

2. move to directory project and Install depedencies

```bash

$ cd github-users

$ yarn install or npm install
```

3. run development server

```bash
$ yarn start or npm start
```
